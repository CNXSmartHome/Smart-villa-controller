# Hardware Specification

## MCU

ESP32-S3-WROOM-1-N16R8

## Power

- Input: 12–24VDC target, tolerate wider range where practical.
- Protection: fuse, reverse polarity, TVS.
- Rails: 5V and 3.3V.

## Interfaces

- RS485 x2
- Relay dry contact x4
- Digital input x8
- Ethernet optional
- Wi-Fi
- USB-C
- RTC
- Buzzer
- Status LED
- Config/reset buttons

## Sensor Reference

MTD062-MB/MTD262-MB class mmWave sensor:
- 9–28VDC sensor supply
- RS485/Modbus capable
- Dry contact output S1/S2
- Relay output ≤30V, ≤2A
