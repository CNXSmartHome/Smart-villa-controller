# Smart Villa OS — Developer Toolkit V2

## Install

Copy `scripts/` and `Makefile` to the repository root.

Then run:

```bash
chmod +x scripts/*.sh
make fix
make doctor
```

## Main Commands

```bash
make doctor
make fix
make build
make test
make bench
make new-component NAME=my_component
make new-driver NAME=my_driver
make release VERSION=v0.1.0
```

## For macOS

Run:

```bash
make setup
```
