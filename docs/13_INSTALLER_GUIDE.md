# Installer Guide

## Installation Flow

1. Mount DIN rail controller.
2. Connect 24VDC supply.
3. Connect mmWave sensor RS485 or dry contact.
4. Connect relay outputs to safe control circuits.
5. Enter provisioning mode.
6. Set admin password.
7. Configure room and delays.
8. Test with non-hazardous load first.
9. Move to real load only after validation.
