# Product Requirement — SVC-100 Pro

## Target Users

- Pool Villa owners
- Hotel operators
- Smart home installers
- Small building automation integrators

## Main Use Cases

- Detect room occupancy with mmWave RS485 or dry contact sensor.
- Turn off lights after no occupancy.
- Turn off air conditioning after no occupancy.
- Read door, leak, smoke, and keycard inputs.
- Control relay dry contacts.
- Operate standalone without Home Assistant.
- Provide local web configuration.
- Support OTA firmware update.

## Minimum V1 Features

- Relay control x4
- Digital input x8
- RS485 Modbus x2
- Local web status page
- Secure provisioning
- Basic automation logic
- OTA update with rollback
