# Project Charter — Smart Villa OS / SVC-100

## Mission

Build a standalone industrial controller for pool villas, hotels, smart homes, and small buildings.

## Product

SVC-100 Pro: ESP32-S3 based controller with RS485, relay dry contacts, digital inputs, local web UI, OTA, and security-first provisioning.

## Success Criteria

- Reliable standalone operation
- Safe relay behavior
- Secure local API
- RS485 mmWave integration
- OTA rollback
- Bench-tested prototype
- Field pilot installation
