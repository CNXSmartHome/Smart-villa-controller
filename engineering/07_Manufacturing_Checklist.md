# Manufacturing Checklist

## Before PCB Order

- Schematic review
- ERC clean
- DRC clean
- BOM review
- Footprint review
- ESP32 antenna clearance review
- RS485 protection review
- Relay isolation review
- Silkscreen terminal review

## Before Field Install

- Firmware version tagged
- Factory test passed
- Enclosure label applied
- Wiring diagram printed
- Installer guide included
