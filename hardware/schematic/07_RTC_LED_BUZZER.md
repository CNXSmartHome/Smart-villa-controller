# 07_RTC_LED_BUZZER — Schematic Sheet

## Purpose

Support installer feedback and optional scheduling.

## Blocks

- RTC: DS3231 or equivalent
- I2C pull-ups
- Status LEDs
- Buzzer driver
- Config button
- Reset button

## Minimum Indicators

- Power LED
- System LED
- Fault LED
- Network LED

## Buzzer Use

- Provisioning mode
- Error code
- Installer confirmation
