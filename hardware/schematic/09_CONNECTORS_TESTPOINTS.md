# 09_CONNECTORS_TESTPOINTS — Schematic Sheet

## Purpose

Define all field wiring and factory test access.

## Connectors

- Power input: VIN / GND
- RS485 Port 1: V+ / GND / A / B
- RS485 Port 2: V+ / GND / A / B
- Relay 1–4: COM / NO / NC
- DI 1–8: DI / COM or shared common
- USB-C
- Expansion header

## Test Points

- VIN
- +5V
- +3V3
- GND
- EN
- BOOT
- UART TX/RX optional
- RS485 A/B optional
