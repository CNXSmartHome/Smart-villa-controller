# 03_RS485 — Schematic Sheet

## Purpose

Provide Modbus RTU communication ports.

## Rev A Target

- RS485 x2 populated
- Optional footprint reserve for RS485 x3

## Per-Port Signals

- RX
- TX
- DE_RE
- A
- B
- GND
- SENSOR_V+

## Recommended Circuit

- 3.3V RS485 transceiver: MAX3485/SP3485
- TVS diode array on A/B
- 120Ω termination jumper
- Bias resistor jumper or footprint
- Pluggable terminal: V+ / GND / A / B

## Firmware Requirements

- Bounded transaction timeout
- Bus lock timeout
- Malformed frame rejection
- Modbus CRC verification
