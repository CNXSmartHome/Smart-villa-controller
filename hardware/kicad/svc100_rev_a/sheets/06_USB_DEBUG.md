# 06_USB_DEBUG — Schematic Sheet

## Purpose

Provide programming, serial log, and factory flashing.

## Circuit Blocks

- USB-C connector
- CC resistors
- USB ESD protection
- D+ / D- to ESP32-S3 native USB
- Optional UART header
- Factory test pads

## Release Gates

- USB orientation verified
- CC resistor values verified
- ESD close to connector
- Boot/programming process documented
